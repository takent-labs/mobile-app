# Takent Mobile App
> [!IMPORTANT]
> **Justificación de la elección tecnológica**
> El proyecto requiere una **"aplicación móvil (Android/iOS)"** que sea intuitiva y adaptable. He optado por **Kotlin Multiplatform** por estas razones:
> - **Entorno oficial:** Cumplo estrictamente con el requisito de "usar Android Studio para el desarrollo", que es la herramienta oficial para este módulo.
> - **Evolución de Java:** Kotlin es la progresión natural de **Java** (que hemos visto en 1º y 2º). Me permite usar un enfoque más moderno y eficiente sin salirme de la base que ya conozco.
> - **Mejora de recursos:** He optado por esta tecnología como una mejora ante los recursos desactualizados que se nos dieron en clase, permitiéndome cumplir mejor con el requisito de "programación sostenible y diseño de software mantenible".
> - **Calidad técnica:** Es la mejor forma de asegurar que la app sea robusta y profesional, aprovechando que las especificaciones permiten que el alumno tome sus propias decisiones.
